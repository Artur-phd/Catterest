# Catterest

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: PostgreSQL
- Other: Git, GitHub

## Features
- Browse a collection of cat images
- Like and save favorite cat images

## Installation
1. Clone the repository: `git clone https://github.com/yourusername/catterest.git`
2. Install dependencies: `yarn install`
3. Set up the database: [Instructions for setting up the database(dbConfig.sql)]
4. Start the server: `yarn node app`
